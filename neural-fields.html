Meta-Learning for Neural Fields

A meta-learning framework for implicit neural representations — combining MAML, NeRV and SIREN decoders with gradient-guided coordinate sampling to efficiently reconstruct images and video from very few pixels.

Year 2024
Type Academic
Status Completed

Neural Field Reconstruction

Overview

Implicit neural representations (INRs) — or neural fields — are neural networks that learn to represent a signal (an image, a video, a sound) as a continuous function of coordinates. Instead of storing pixel values directly, the network learns a mapping from coordinates to signal values, essentially compressing the signal into its weights.

This project implements a full meta-learning pipeline on top of neural fields: rather than training each network from scratch per signal, a meta-learner finds an initialisation that can be quickly adapted to any new signal in just a handful of gradient steps. The codebase supports two decoder architectures (SIREN and NeRV), multiple data modalities (images, video, audio, manifold), and a gradient-guided sampling strategy that intelligently selects which coordinates to use during inner-loop adaptation.

The Problem

Training a separate neural field from scratch for every signal is expensive and slow. Meta-learning offers an elegant solution: if we can find a good shared weight initialisation across many signals, each new signal only needs a small number of inner gradient steps to fit — dramatically reducing per-signal compute.

A second challenge is what coordinates to use during adaptation. Using all coordinates is slow, and random subsampling wastes budget on easy, low-information pixels. The question is whether we can use gradient information to prioritise the coordinates where adaptation will learn the most.

Approach

The project is structured around three main components that can be combined modularly.

  • Meta-learning with MAML — The outer loop trains a shared initialisation using Model-Agnostic Meta-Learning (MAML). Each inner loop takes a small number of gradient steps to adapt the shared parameters to a specific signal, and the outer loss is computed on the full signal after adaptation. A bootstrapped variant of MAML (maml_bootstrap_param) was also implemented and evaluated.
  • Two decoder architectures — SIREN uses sinusoidal activations with a carefully designed initialisation scheme, making it well-suited for signals with fine structure. NeRV is a convolutional video decoder that takes a frame index as input and outputs the full frame via a stack of upsampling blocks — a fundamentally different approach that is much faster for video. Both are wrapped in a MetaWrapper that handles coordinate generation, sampling, and MSE loss computation uniformly across modalities.
  • GradNCP coordinate sampling — Instead of random subsampling, GradNCP selects the coordinates where the gradient of the last-layer weights is largest. For SIREN this is computed as the outer product of the penultimate feature vector and the reconstruction error; for NeRV it uses the Jacobian approximation via tanh derivatives. Only the top fraction (controlled by data_ratio) of coordinates are used in the inner loop, focusing adaptation budget on the most informative locations.
  • SPO fine-tuning stage — After meta-training, a second training phase (Split Parameter Optimisation) fine-tunes only the final layer of the network per-signal while keeping the shared trunk frozen. This achieves strong single-signal accuracy without disturbing the meta-initialisation.
  • Experiment tracking with W&B — All runs were tracked with Weights & Biases, logging inner loss, batch outer loss, and average outer loss at each step. Multiple runs were conducted with different MAML configurations to compare convergence behaviour.

Results

The meta-trained models successfully converge to shared initialisations that can be quickly adapted per signal. PSNR (Peak Signal-to-Noise Ratio) is used as the primary reconstruction metric — evaluated both inside (in-context) and outside (out-of-context) the inner adaptation loop. The GradNCP sampling strategy consistently yields stronger adaptation than random sampling at the same coordinate budget, confirming that gradient-guided selection of coordinates is worthwhile even with as little as 25% of all pixels.

Reflections

The most interesting design tension in this project is the tradeoff between the two decoder architectures. SIREN is elegant and general — one network handles any modality through coordinate inputs — but is slow for video because it must process every spatial coordinate individually. NeRV flips the design: it treats frames as atomic units and uses convolutional upsampling to decode from a single time index, making it much faster but sacrificing the modality-agnostic property.

GradNCP's coordinate selection is computationally cheap at inference time once the penultimate features are available, and the improvement over random sampling shows that the gradient signal is genuinely informative. An interesting extension would be to combine adaptive sampling with uncertainty estimates from the network itself — selecting coordinates where the model is both uncertain and the gradient is large.
